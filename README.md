# Africa Knowledge Vault (AKV)

## Mission

Africa Knowledge Vault (AKV) est une infrastructure panafricaine de préservation numérique et de diffusion des connaissances visant à protéger et rendre accessibles les savoirs scientifiques, culturels, linguistiques et historiques de l'Afrique.

## Vision

Créer la mémoire numérique permanente de l'Afrique.

## Objectifs

- Préserver les connaissances africaines
- Archiver les thèses et mémoires
- Sauvegarder le patrimoine culturel
- Préserver les langues africaines
- Faciliter la recherche scientifique
- Alimenter une future IA fondée sur les connaissances africaines

## Fonctionnalités

- Bibliothèque numérique
- OCR multilingue
- Recherche plein texte
- Elasticsearch
- OAI-PMH
- Dépôt scientifique
- Archives culturelles
- Archives orales
- API REST
- Préservation numérique

## Horizon

- 10 millions de documents
- 54 pays africains
- Plusieurs centaines d'institutions partenaires
